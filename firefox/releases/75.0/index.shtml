Firefox 新鮮事

  • 全新功能 今天的發行帶來許多的改進，幫助你搜尋更加聰明、快速。打得更少，尋找更多，使用Firefox 翻新的網址列：
    • 專為小螢幕筆電最佳化，專注、簡潔的搜尋體驗。
    • 選擇網址列時將會出現熱門網站。
    • 改善了搜尋字詞建議的可讀性，凸顯新的搜索關鍵字。
    • 搜尋建議加入 Firefox 常見問題的解決方案。
    • 在 Linux 平台上，點擊網址列與搜尋列的行為現在與其他桌面平台相同：單擊選擇全部，但不進行快速複製（Primary Selection），雙點會複製單詞，三點則選取全部並進行快速複製。
  • 全新功能 Firefox 會在本地暫存所有 Mozila 已知的網路憑證機構所簽發的憑證，這將會改善錯誤配置伺服器的 HTTPS 相容性與提高安全性。
  • 全新功能 Firefox 在 Flatpak 上架了。Flatpak 是一個在更容易在 Linux 上安裝與使用 Firefox 的方式。
  • 全新功能 對於我們的 Windows 使用者，Direct Composition 目前已被整合，以改善效能，同時有益於 Intel 顯示卡的 Windows 10 筆記型電腦預載 Firefox。
  • 修正 各種安全性修正。
  • 企業版 設定 security.osclientcerts.autoload 為 true ，以啟用對本地作業系統憑證的實驗性支援。
  • 企業版 可使用企業政策來停止透過 DNS over HTTPS 的可信任供應商（TRR, Trusted Recursive Resolver）解析網址。
  • 開發功能 請參閱 Mozilla Hacks 部落格上的 Firefox 75: 四月新氣象 一文。
  • 開發功能 網路資訊面板現在可以顯示封鎖的資源，讓開發者充分感受到內容封鎖功能的威力，還有這版能讓我們為所有使用者送上持續進化的加強型追蹤保護功能。
  • 開發功能 在 <img> 使用 loading 屬性來節省頻寬與減少瀏覽器記憶體使用量。預設值 eager 會讓圖片立即載入，而 lazy 延遲載入圖片，直到進入 viewport 的範圍內。
  • 開發功能 主控台的即時求值功能，使開發者可以比以往更快找出與修正錯誤。只要無副作用（side-effect free）的表達式一輸入到網頁主控台中，就可以即時預覽結果。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
